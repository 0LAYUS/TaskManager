# TaskManager 📋

Aplicación web de gestión de tareas personales desarrollada como proyecto de ejemplo para la **Actividad 4 - Sistemas de Control de Versiones** del módulo Mantenimiento de Software.

## Descripción

TaskManager es una aplicación web sencilla que permite a los usuarios:
- Agregar tareas con título, descripción y prioridad
- Marcar tareas como completadas
- Filtrar tareas por estado (todas, pendientes, completadas)
- Eliminar tareas
- Persistencia de datos en localStorage

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
